# 💼 Personal Portfolio Website

A fully responsive personal portfolio website built using **HTML, CSS, and JavaScript**, designed to showcase my skills, projects, and professional background as a B.Tech Computer Science graduate.

🔗 **Live Demo:** [Add your deployed link here]

---

## 📌 Overview

This portfolio highlights my work in software development, UI/UX design, and machine learning. It features a clean, modern interface with smooth navigation, interactive sections, and a functional contact form — built entirely from scratch without any front-end frameworks.

---

## ✨ Features

- 🎨 Clean, modern UI with custom design and animations
- 📱 Fully responsive layout — optimized for mobile, tablet, and desktop
- 🍔 Mobile slide-in navigation menu
- 🗂️ Interactive tabbed sections (Skills / Experience / Education)
- 🖼️ Dynamic portfolio/work showcase with hover effects
- 📩 Functional contact form integrated with Google Sheets (via Apps Script)
- ⚡ Smooth scroll behavior and hover transitions throughout

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Page structure & semantic markup |
| CSS3 | Styling, layout (Flexbox/Grid), responsive design |
| JavaScript | Interactivity — tab switching, mobile menu, form handling |
| Font Awesome | Icons |
| Google Apps Script | Contact form backend (Google Sheets integration) |
